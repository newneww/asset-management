# ระบบจัดการทรัพย์สินอุปกรณ์ (Asset Management)

เว็บแอปติดตามการยืม/ติดตั้ง/ตรวจนับอุปกรณ์ที่บริษัทให้ร้านค้ายืมไปใช้งาน
**ทั้งเว็บทำงานอยู่ใน Google Apps Script** และ **เก็บข้อมูลใน Google Sheets**
— ไม่ต้องมีเซิร์ฟเวอร์ ไม่ต้องโฮสต์ที่อื่น ไม่ต้องใช้ npm/GitHub/Cloudflare

> ✅ **วิธีใช้จริงมีแค่ 2 ไฟล์:** `apps-script/Code.gs` + `apps-script/Index.html`
> วางใน Apps Script ของ Google Sheet แล้ว Deploy — **URL `/exec` คือตัวเว็บเลย**

## ภาพรวม

- **หน้าเว็บ (UI):** `apps-script/Index.html` — Apps Script เสิร์ฟให้เอง (HtmlService)
- **ฐานข้อมูล:** Google Sheets 4 แท็บ — `Shop Master`, `Asset Master`, `Log & History`, `Audit`
- **ตัวประมวลผล (API):** `apps-script/Code.gs`
  - เลขที่ใบยืมอัตโนมัติ (`LN-YYYY-NNNN`)
  - สร้าง **สัญญายืม PDF → เก็บใน Google Drive**
  - แดชบอร์ด/สรุปยอด, ส่งออก CSV
  - เข้าสู่ระบบด้วยชื่อผู้ใช้/รหัสผ่าน + สิทธิ์ 3 ระดับ

## บทบาทผู้ใช้

| Role | ใช้งานหลัก | สิทธิ์ |
|------|-----------|--------|
| `technician` (ช่าง) | มือถือ/แท็บเล็ตหน้างาน | ทำใบยืม+เซ็นสัญญา, แจ้งย้าย/คืน, ตรวจนับ, ดูประวัติร้าน |
| `admin` (แอดมิน) | เดสก์ท็อป | จัดการทุกตาราง, อนุมัติการเคลื่อนย้าย |
| `executive` (ผู้บริหาร) | เดสก์ท็อป | ดูแดชบอร์ด/รายงาน + ส่งออกไฟล์ |

## วิธีติดตั้ง (ทำครั้งเดียว ~5 นาที)

1. เปิด **Google Sheet** ที่จะใช้เป็นฐานข้อมูล → **Extensions → Apps Script**
2. **ไฟล์ `Code.gs`** — ลบของเดิม แล้ววางเนื้อหาจาก `apps-script/Code.gs`
3. **เพิ่มไฟล์ HTML** — กด **+ → HTML** ตั้งชื่อว่า **`Index`** (ต้องชื่อนี้เป๊ะ)
   แล้ววางเนื้อหาจาก `apps-script/Index.html`
4. (แนะนำ) เพิ่ม manifest: ⚙️ **Project Settings → Show "appsscript.json"** แล้ววาง `apps-script/appsscript.json`
5. เลือกฟังก์ชัน **`setup`** ในแถบด้านบน → **Run** → กด **Allow** อนุญาตสิทธิ์
   - สร้างแท็บทั้ง 4 + ผู้ใช้ **`admin` / รหัสผ่าน `admin1234`**
6. **Deploy → New deployment → Web app**
   - *Execute as:* **Me**
   - *Who has access:* **Anyone**
   - **Deploy** → คัดลอก **Web app URL** (ลงท้าย `/exec`)
7. เปิด URL `/exec` นั้นในเบราว์เซอร์ → **ได้หน้าเว็บเลย** → เข้าสู่ระบบ `admin` / `admin1234`
   → **เปลี่ยนรหัสผ่านทันที** (ดูหัวข้อถัดไป)

### แก้ไขโค้ดแล้วอัปเดตเว็บ (ให้ URL เดิมไม่เปลี่ยน)
**Deploy → Manage deployments → (ดินสอ ✏️) Edit → Version: New version → Deploy**

### เพิ่ม/เปลี่ยนรหัสผ่านผู้ใช้ (ช่าง/แอดมิน/ผู้บริหาร)
แก้ค่าในฟังก์ชัน `createUser()` ใน `Code.gs` (username / ชื่อ / role / password) แล้วกด **Run**
รันซ้ำด้วย username เดิม = รีเซ็ตรหัสผ่าน (รหัสถูก hash ด้วย SHA-256 ไม่เก็บ plaintext)

## โครงสร้างข้อมูล (แท็บใน Google Sheet)

- **Shop Master** — ร้านค้า
- **Asset Master** — ทรัพย์สินรายชิ้น (สถานะ: อยู่ในคลัง/ถูกยืม/ชำรุด/สูญหาย)
- **Log & History** — เหตุการณ์ยืม/ย้าย/คืน (เก็บ items เป็น JSON + ลิงก์สัญญา PDF)
- **Audit** — การตรวจนับประจำปี

> คอลัมน์ทั้งหมดกำหนดที่ `apps-script/Code.gs` (ค่าคงที่ `HEADERS`) — แก้ที่จุดเดียว

## หมายเหตุ: เวอร์ชัน React (ไม่จำเป็นต้องใช้)

โฟลเดอร์ `src/` มีเวอร์ชัน React + Vite (เผื่ออยากโฮสต์แยกในอนาคต) — **ข้ามได้เลย**
สำหรับการใช้งานจริง ใช้แค่โฟลเดอร์ `apps-script/` ก็พอ
